## 📖 使用说明

### 🎯 基本操作
1. **添加命令** - 在顶部输入框中输入命令名称和内容，选择目标文件夹
2. **快速搜索** - 使用顶部搜索框或按 `Ctrl+K` 快速定位命令
3. **编辑命令** - 点击命令卡片的"编辑"按钮，支持多行编辑
4. **一键复制** - 点击"复制"按钮，命令自动复制到剪贴板
5. **文件夹管理** - 点击侧边栏的"➕"按钮创建文件夹，支持重命名删除

### ⌨️ 快捷键操作
- `Ctrl+K` - 聚焦搜索框，快速搜索
- `Ctrl+N` - 新建命令，快速添加
- `Ctrl+Shift+N` - 新建文件夹
- `Ctrl+Enter` - 添加/保存命令
- `Escape` - 取消编辑/关闭操作

### 🔥 高级特性
- **🖱️ 全卡片拖拽** - 命令内容区域也可拖拽，不限于边缘
- **📁 智能分类** - 拖拽命令到文件夹自动分类
- **💾 数据管理** - JSON格式导入导出，完整数据备份
- **🎨 视觉反馈** - 操作状态实时显示，保存成功明确提示

## 🛠️ 技术栈

### 💻 核心技术
- **前端框架**: 原生 HTML5 + CSS3 + Vanilla JavaScript ES6+
- **设计系统**: Apple Glass Morphism + CSS backdrop-filter
- **数据存储**: localStorage 持久化存储
- **架构模式**: ES6 Class + 模块化设计 + 事件委托

### 🎨 样式技术
- **CSS变量系统**: 统一的设计token管理
- **布局方案**: Flexbox + CSS Grid 混合布局
- **动画效果**: CSS Transitions + Transforms + GPU加速
- **响应式设计**: 移动优先 + 断点媒体查询

### 🚀 现代特性
- **毛玻璃效果**: backdrop-filter + 多层阴影
- **鼠标跟随**: CSS自定义属性 + 实时位置更新
- **拖拽API**: HTML5 Drag & Drop + 事件委托
- **异步操作**: async/await + Promise + 错误处理
